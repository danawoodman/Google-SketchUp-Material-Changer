Google SketchUp Material Changer by danawoodman

A simple Google SketchUp plug-in that allows you to select any number of entities in your SketchUp model and change either the face material, back material or both. Works recursively on all sub-entities in the selection.

Dependencies

SketchUp 8+ (should also work for SketchUp 6+)

Install

Copy the file `SU_MaterialChanger.rb` into your SketchUp plugin directory: On a **Mac** it is usually here: Macintosh HD/Library/Application Support/Google SketchUp X/SketchUp/ On **Windows** it should be here: C:\Program Files\Google\Google Sketchup X\Plugins\ .. where `X` is the version of SketchUp you are running.
